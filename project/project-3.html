마이플레이스

마이플레이스는 주변의 맛집, 카페 등을 찾을 수 있는 어플 입니다. 몇몇 주변 장소가 카드 형태로 표시되며 해당 장소를 선택해 리뷰를 작성할 수 있습니다. 또한 주변 장소에 대한 다른 사람들의 리뷰를 확인할 수 있습니다.

음식점 정보는 공공데이터포털 API를 사용했습니다.

주요 기능

  • 회원가입
  • 로그인 & 로그아웃
  • 음식점 정보 탐색
    • 위치와 장소 키워드를 입력하여 원하는 음식점 찾기
    • 현재위치에 있는 음식점 찾기
  • 장소찜하기 및 평가/리뷰
  • 다른 사용자들 리뷰 모아보기
  • 회원정보 수정

기술/언어

  • Java
  • Android

라이브러리

  • Google Map
  • Retrofit
  • Glide
