WEBSITE & WEB APP / (주)스터디짐

STUDYGYM

Google Play

스터디짐은 강사가 웹사이트에 강의와 문제를 등록하면, 학생이 웹/앱으로 해당 강의를 수강하고 시험을 제출하는 학습 플랫폼입니다.

주요 업무 (퍼블리싱)

  • 학생/학부모 하이브리드 앱 리뉴얼 및 유지보수
  • 학생 웹사이트 리뉴얼 및 유지보수
  • 원장웹 사이트(시험문제 출제, 강의 업로드, 학생관리 등) 제작 및 유지보수
  • 학원 홍보 웹사이트 제작
  • 관리자 웹사이트 유지보수

기술/언어

  • HTML & CSS
  • JavaScript

PR

대규모 프로젝트를 0부터 100까지 맡아 진행하면서, 퍼블리싱을 어떤 방향으로 해야 하는가에 대한 고민이 많았습니다. 기존에는 페이지 단위로 마크업을 하고, CSS를 적용하는 방식이었습니다. 하지만 기존 방식은 다른 페이지에 이전과 비슷한 디자인이 존재하면 또다시 새로운 CSS를 코딩해야 하는 비효율적인 상황이 생겼습니다. 그래서 기존 방식이 아닌 컴포넌트 단위로 퍼블리싱을 했습니다. 그 결과, 전체 코드의 양이 30%가량 줄었고, 유지보수 또한 쉬워졌습니다. 이러한 사례처럼 저는 기존 방식만을 고집하기보다는 효율적인 방법을 찾고자 다양한 방법들을 탐색합니다.
